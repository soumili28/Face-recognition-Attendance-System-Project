### 📸 Face Recognition Attendance System

A smart and automated attendance system built using Python, OpenCV, and Face Recognition that detects faces in real-time through a webcam and marks attendance automatically in a CSV file.

This project eliminates manual attendance, prevents proxy attendance, and improves accuracy using computer vision and machine learning techniques.

### 🚀 Features
Real-time face detection using webcam
Face recognition using image encoding
Automatic attendance marking
Date and time-based attendance logging
Prevents duplicate attendance entries
CSV-based attendance record storage
Simple, lightweight, and easy to use
